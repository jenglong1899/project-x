本项目建立在以下理念之上：
AI 是锯齿状智能，同一个 AI 能够解决很难的问题，但有时候简单问题都解决不了。

本项目能给您提供：
- 安全性：
  - 龙虾将在沙盒中运行，您可以控制它能修改哪些文件，读取哪些文件
  - 龙虾不会持有原始 API 密钥
- 可监督性：在网页端我们提供了“在下一轮工具调用后暂停”功能和“立刻终止工具调用”功能。
- 主动性：基于定时任务实现
- 接入 IM 平台：Slack、飞书
- 比原版龙虾更加优秀的记忆效果
- 更少的 bug ：本项目的所有代码均经过人类工程师审核
- 更好的表现，更低的成本：本项目的指令（提示词）力求精简零废话

下一期推出的功能：
- 多智能体：由于 AI 的发挥不稳定性（锯齿状智能），
- 操作电脑（Computer use）
