模仿人脑的记忆机制的必要性？自然界的代码都是屎山，人的记忆就有不靠谱的地方，回想一件事的时候容易被误导，你想一天记1000个单词却记不住。还有必要模仿人脑吗？从第一性原理分析，llm和人脑并不完全相同，因此模仿人脑的机制未必有效。

目前可以接受的标准：达到和人类类似的水平
- 允许少量遗忘

可以轻易超越人脑的地方：
- 想记住什么就记住什么

不过屎山也不意味着没有可以借鉴的地方。
最理想的状态是你能操控自己记住什么、忘记什么
如果所有信息都会被记住，而无法遗忘，那对生活是有害的，不如“默认遗忘，只有新奇体验、重复见过的才会记住”
